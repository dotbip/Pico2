# DotBip-for-Raspberry-Pi-Pico-2-connected-to-a-Pimoroni-Display-Pack-2.0
If you are a security minded maximalist and want to encrypt and decrypt a Binary Grid of Seed Phrase Words (Dot Map) then please download and then load this code via Thonny to your Raspberry Pi Pico 2 connected to a Pimoroni Display Pack 2.0" and enjoy an offline air-gapped experience.
